Getting started

From an empty directory to a validated AI-agent artifact in five commands.

1. Install

aikido needs Python 3.10+. Install it in editable mode from a clone of the repo:

$ git clone https://github.com/edwardmonteiro/aikido
$ cd aikido
$ pip install -e .
$ aikido --help

For development (tests, formatting, type-checking):

$ pip install -e ".[dev]"
$ pytest tests/

2. Initialize a project

aikido init scaffolds the full knowledge tree, writes the default templates, contracts, and control-plane files, and (unless you pass --no-git) initializes a git repository with a first commit.

$ aikido init my-company
$ cd my-company
Idempotent. Re-running init never overwrites files you've edited — it only fills in what's missing.

3. Capture your day

The guided daily capture asks a handful of questions — day type, what you shipped, blockers, a customer moment, and a one-sentence takeaway — then auto-tags the entry from its content and saves it under the current ISO week.

$ aikido daily

# capturing a past day?
$ aikido daily --retroactive 2026-07-14

Saved to 01_weekly_cadence/2026-W29/2026-07-14_daily.md.

4. Build the week

aikido weekly aggregates every daily log in the week into four rollups — decisions.md, blockers.md, customer_signals.md, daily_summary.md — and then transpiles every agent that has both a template and a contract.

$ aikido weekly --week 2026-W29

5. Transpile a single agent

To build one agent on demand — optionally scoped to a focus area that pulls in extra domain context:

$ aikido build_agent founder_weekly_review --week 2026-W29
$ aikido build_agent founder_weekly_review --week 2026-W29 --focus-area product

The result is a flat Markdown system prompt in 06_dist/ with all includes resolved, all contract outputs present, and constraints checked:

# Founder Identity
...
## This Week's Context (2026-W29)
### Decisions
- [2026-07-14] Shipped: Shipped SSO and new pricing page
...
## Output Format
Return ONLY JSON:
{
  "next_week_priorities": ["string", "string", "string"],
  "decision_recommendation": "string",
  "risk_flag": "string",
  "investor_snippet": "string"
}

Keep it healthy

Validate the knowledge base at any time — broken includes, undefined variables, contract mismatches, and stale files are all reported:

$ aikido validate
$ aikido dashboard        # system-health overview
